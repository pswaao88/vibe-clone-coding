# 추가된 기능 목록

## 🎉 새로 구현된 기능들

### 1. 상품 업로드 기능 ✅
- **경로**: `/products/upload`
- **기능**:
  - 상품 제목, 설명, 가격, 카테고리 입력
  - 이미지 업로드 (다중 이미지 지원)
  - 상품 등록 및 마켓에 자동 표시

### 2. 상품 검색 및 필터링 ✅
- **위치**: 마켓 페이지 (`/market`)
- **기능**:
  - 상품 제목/설명 검색
  - 카테고리별 필터링 (전자제품, 의류, 가구, 도서, 기타)
  - 실시간 검색 결과 업데이트

### 3. 내 상품 관리 페이지 ✅
- **경로**: `/my-products`
- **기능**:
  - 판매 중인 상품 목록 조회
  - 상태별 필터링 (전체, 판매중, 예약중, 판매완료)
  - 내가 등록한 모든 상품 관리

### 4. 채팅 기능 완성 ✅
- **경로**: `/chat`
- **기능**:
  - 채팅방 목록 조회 (구매자/판매자 모두)
  - 실시간 메시지 송수신
  - 거래 완료 시 자동 채팅방 생성
  - 메시지 읽음 상태 관리

### 5. 포인트 충전 기능 ✅
- **위치**: 지갑 페이지 (`/wallet`)
- **기능**:
  - 포인트 충전 폼
  - 사전 설정된 금액 선택 (10,000원, 30,000원, 50,000원, 100,000원)
  - 직접 금액 입력
  - Cloud Function을 통한 안전한 충전 처리

### 6. 프로필 수정 페이지 ✅
- **경로**: `/profile`
- **기능**:
  - 이름(displayName) 수정
  - 프로필 이미지 URL 설정
  - 이메일 및 포인트 조회 (읽기 전용)

## 🔄 개선된 기능

### 상품 상세 페이지
- 거래 완료 후 자동으로 채팅방 생성
- 구매 완료 시 채팅 페이지로 이동

### 헤더 네비게이션
- "내 상품" 메뉴 추가
- 프로필 링크 추가

### 마켓 페이지
- 상품 등록 버튼 추가 (로그인 사용자만)
- 검색 및 필터 UI 개선

## 📁 새로 추가된 파일

### Frontend
- `src/features/market/pages/ProductUploadPage.jsx` - 상품 업로드 페이지
- `src/features/market/pages/MyProductsPage.jsx` - 내 상품 관리 페이지
- `src/features/auth/pages/ProfilePage.jsx` - 프로필 수정 페이지

### Backend
- `functions/src/charge.js` - 포인트 충전 Cloud Function

## 🚀 사용 방법

### 상품 등록
1. 마켓 페이지에서 "상품 등록" 버튼 클릭
2. 상품 정보 입력 및 이미지 업로드
3. "등록하기" 버튼 클릭

### 포인트 충전
1. 지갑 페이지 접속
2. "포인트 충전" 버튼 클릭
3. 금액 선택 또는 직접 입력
4. "충전하기" 버튼 클릭

### 채팅하기
1. 상품 구매 완료 시 자동으로 채팅방 생성
2. 채팅 페이지에서 채팅방 선택
3. 메시지 입력 후 전송

### 내 상품 관리
1. 헤더에서 "내 상품" 클릭
2. 상태별 필터로 상품 조회
3. 상품 클릭하여 상세 정보 확인

## 🔐 보안

- 모든 포인트 관련 작업은 Cloud Functions에서 처리
- 사용자는 본인의 포인트만 충전 가능
- 상품은 본인 것만 수정/삭제 가능
- 채팅방은 참여자만 접근 가능
